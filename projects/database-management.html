Olympic Dataset: Database Design & Visualization Dashboard

SQL + ERD + Normalization + APEX Dashboard + Medal Analysis + Gender/Sport Comparisons

Project Overview

This group project explores historical Olympic performance trends using a relational schema and a fully interactive APEX dashboard. The database was derived from a 2023 Kaggle Olympic dataset containing 70,000 original records and reduced to 5,642 normalized rows after cleaning and sampling. According to the report, the ERD model connected 5 tables (Athlete, Team, Location, Event, Participation), enabling interactive analytics of medals, country dominance, athlete age distribution, and sport-specific gender patterns .

Visual outputs include medal leader charts, gender-based comparisons, event participation breakdowns, and dashboards that allow filtering of athlete demographics and medal trends across Olympic years. The analysis reveals U.S. dominance with 28% of total medals in the sampled set and highlights notable sport-level gender variations .
